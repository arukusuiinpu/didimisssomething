# Did I Miss Something Host

This project is a child repository for [Did I Miss Something?](https://github.com/arukusuiinpu/didimisssomething)

That project uses the **latest release** of this one to populate the game with **mods and configs**.

# How to update my mods?

Ohhhh that's simple.

All you need to do is to make a **new release** and make sure to mark it as the **latest release**.
